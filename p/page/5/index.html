git clone提速方案

由于一些神奇的原因，git在国内速度较慢。本文章针对已经有SSR的情况。

装好SSR之后开socks5代理。显然用浏览器是没问题的，但是git的三种协议（http、ssh、git）都默认不走代理，需要分别整活（过于繁琐，有可能还是直接把飞机开到路由器上）。
