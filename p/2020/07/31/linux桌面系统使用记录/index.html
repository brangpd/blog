Ubuntu

目前环境

Ubuntu 20.04.1 amd64 GNOME 3.36.3 X11

日常相关

社交

Deepin移植了一些软件, 包括QQ, 微信, TIM(不支持Linux的屑腾讯), 百度网盘(已有官方Linux版本, (别用百度云了好吗)): http://packages.deepin.com/deepin/pool/non-free/

其中微信没有尝试过.

QQ, TIM以及QQ light版: TIM似乎已经是个搞笑的产品了; QQ light功能有限; 使用QQ.

在挂了代理的情况下, QQ系软件可能出现下载文件速度相当慢, 图片加载不出来的情况: 在登录QQ的界面找到设置, 把代理加上去, 选中选择框后键盘上下键可做选择, 原因是默认将主窗口显示在最上层. QQ登录后在设置里把主面板总是位于上方一项去掉.

远程控制

本来在用TeamViewer, 然而这个玩意报了个商用判定, 改用向日葵. 支持Windows, Linux, iOS, 目前情况还行.

会议

腾讯会议目前无解. 使用向日葵从iPad转播电脑.

输入法

改掉ibus, 但是不能把它删了, 因为是系统组件.

选择fcitx, 中文fcitx-rime, 日文fcitx-mozc.

更改字体和大小, 不是在rime的配置中更改而是在fcitx的配置中更改.

fcitx的小图标里面选项无法展开, 设置fcitx调出输入栏在输入栏选择.

GNOME Extensions & Tweaks

  • Dash To Panel (在下方显示一排打开的窗口比较合适)
  • Tray Icons (主要是把QQ的那个Wine System Tray弄掉, 然而不能悬浮预览消息)
  • User Theme (Vimix主题, Papirus图标)
