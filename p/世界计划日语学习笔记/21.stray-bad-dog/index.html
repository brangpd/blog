1.感じる成長

  • まだまだ伸びしろはありそうだ
    • のびしろ：有发展的潜能
  • バックステージ
  • 浮かれる
  • 踏ん切りがつく：決断
  • ミスる
  • 未知数
  • いい波がきてるんだ
  • 気を引き締める
  • サンプラー
  • 許してあげなくもないかな
  • ヘラヘラ：傻笑，喋喋不休的态度

2.謎の実力者

  • 練習した分がちゃんと出てるって感じ
  • 朝練（あされん）やった甲斐は十分あった
  • 楽屋（がくや）
  • この前の「あいつ」といい勝負なんじゃないか
  • かなり歌えるようだ
  • 公言
  • ここら
  • 海外で修行（しゅぎょう）
  • 気のいいやつ
  • 頭も回るし、客を盛り上げるセンスも頭一つ二つ抜けてた
    • 頭が回る：头脑灵活
    • 頭一つ抜ける：他よりも優れている、他を引き離すように卓越（たくえつ）しているさま、群（ぐん）を抜いて優れている・目立っているさま、抜（ぬ）きんでているさま
  • 息巻いていた：激しく怒る、勢い込んで言う
  • 可愛がる連中
  • 腕を上げてる
  • 予定空けとけ
  • 確認しときてえ：しておきたい
  • 人だかりが出来た
  • 飛び入りで参加する
    • 飛び入り：予定していた以外のものが参加する
  • 最後に新に歌われたら、全部持ってかれるだろ
    • 持っていかれる
  • 誰待ち
  • 知り合いのハコで
    • ハコ=小規模のライブハウス
  • そこまで気合入ってるようには
    • 気合：精神を集中してかかるときの勢い、また、その声
    • 気合を入れる：鼓劲
    • 気合を合わせる
    • 気合をかける（声出し）
  • ああだこうだ考える
  • 話は、あいつの歌を聞いてからだ
  • 願ったり叶ったり

3.衝突

  • 飯屋（めしや2）
  • チョコパフェ
  • これだけは外せないんだ
    • 外す：逃す、错过
  • お話させてもらって
  • むりやり連れてかれてさ
  • けんさんが目をかけてるほどのチーム
    • 目をかける：注目して贔屓にする
  • 良い線まで行ける
  • 舐めんじゃない
    • 舐める：馬鹿にする
  • まだ粗いところが目立つけど
  • やりあう：言い争う
  • のこのこ帰れ
    • のこのこ：厚かましくも、不知廉耻装作没事
  • 沸かす
  • 前に決着つけてやる
    • 決着をつける：結果が出る
  • 攻める

4.そんなの、当たり前

  • しっくり：融洽、符合、好好地
  • 思う存分
  • 聞かせてあげる
  • 凄（すさ）まじい
  • 突き刺さる
  • にしても
  • 思い入れがある：執心
  • 受け売り：复述他人的说法
  • 突っ込んできた車にはねられて
  • 一命はとりとめたけど
    • 一命を取り留める
  • 麻痺が残って
    • 麻痺：麻痹或瘫痪
  • ノリで：調子に乗って
  • 柄にもなく
    • がらにない：与身份不匹配
  • その矢先だった
  • プレイヤー
  • 荒削り
  • 努力を重ねる

5.向けられた背中

  • 座り込んで
  • 敗北を*糧（かて）*にして
  • 食らいついてきた
    • 食らいつく=くいつく：噛み付く、とって離れない
  • 余計にもったいない
  • 至らないところがあった
  • 肩を並べる
  • 力をつける：腕前を上げる
  • シフトを削る
  • この程度のどこが大変なの
  • 手も足も出ない：相手に力が及ばずどうしょうもない
  • まずない
    • まず：最初に；おそらく；一応
  • むしゃくしゃ：いらいら
  • 平謝り
  • 俺達のところには謝りに来てない
  • 合わせる顔
  • 上り詰める：登り切る
  • 場数を踏む
  • 経験値を稼げる
  • 思いっきりふらついた
    • ふらつく：站不稳、闲逛、思考犹豫
  • なんてことない=大したことはない

  • 無茶だろうが何だろうが
  • 程遠い
  • 壁にぶつかる
  • がむしゃら：拼命
  • やりようがある
    • やりよう=やり方
  • おしゃれな曲
  • サンプリング
  • 聞き入（い）る
  • と思ったら
  • 負けたからってあんなに無茶なことをするかって言うと…
    • って言うと、そうじゃなく
  • もしかすると
  • 焦っている
  • 父さんのことを吹っ切って
  • 出会った頃よりずっと成長してる
  • 夢を追う
  • 頼らせてあげる
  • 戻ってきたことだし
  • さくっとあきとに連絡とって
  • こき下ろす：けなす、贬低
  • 客にいちゃもんつけられたり
    • いちゃもんをつける：文句言う
  • そのたびに
  • 覇気がある：野望
  • 努力って言葉で終わらせていいレベルじゃなかった
  • クラシックやってたってだけあって、当時から音楽のセンスも実力もあった
  • 騒がれてきた
  • その反面
  • 粋がる・意気がる：自以为了不起
