CentOS 8 NFS配置

安装NFS

1

yum -y install nfs-utils

服务端

修改/etc/exports:

1

/data *(rw,sync,no_subtree_check,no_root_squash)

说明:

  • rw/ro: 读写/只读
  • sync/async: 同步/异步
  • no_subtree_check/subtree_check: 用途未知
  • no_root_squash/root_squash: 对于root用户登录是否直接转成匿名用户
